# M-1 — 0.3.0 mit visueller Test-Ebene

Eine Design-Bibliothek, deren Tests nur HTML-Strings prüfen, testet das Falsche: Ob eine Komponente
*aussieht* wie vorgesehen, steht nicht im Markup.

**Fertig, wenn:** jede Komponente je Ansichtsgröße als Bild erfasst wird und eine Abweichung den
Lauf rot macht — nicht erst das Auge beim nächsten Durchsehen.
